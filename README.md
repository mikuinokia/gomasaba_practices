# gomasaba_practices

## 概要
プログラミングの基礎を学ぶための練習リポジトリ。  
現在はPHPのみに対応している。

## 必要知識
このリポジトリを利用するための最低限のスキルは以下

- Linux基礎知識
  - [Linux標準教科書](https://linuc.org/textbooks/linux/)程度の知識
- プログラミング言語知識
  - ローカルでのプログラムの実行や簡単なスクリプトの実装知識

### PHPで取り組む場合

- composerの基礎知識
- phpcsの基礎知識
- PHP文法の基礎知識

## algorithm
アルゴリズムを練習するためのディレクトリ。

- 問題
- テストパターン
- テスト実行スクリプト

が用意されている。
